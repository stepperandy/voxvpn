# VoxVPN User Guide

## Getting Started

### Download & Install

1. Visit [voxvpn.com/download](https://voxvpn.com/download)
2. Download for your platform (Windows, macOS, Linux, iOS, Android)
3. Run the installer and follow prompts

### Create Account

1. Open VoxVPN app
2. Click "Sign Up"
3. Enter email and create password
4. Verify email address
5. Choose subscription plan

### Connect to VPN

1. Open VoxVPN
2. Select a server from the list
3. Click "Connect"
4. Wait for connection confirmation
5. Your traffic is now encrypted

## Choosing a Server

### By Country
- Select your desired country from the dropdown
- Servers are sorted by load (lowest first)

### By Performance
- Green = Low load, excellent speed
- Yellow = Medium load, good speed
- Red = High load, slower speed

### Fastest Server
- Click "Auto-Select Fastest"
- App automatically connects to best available server

## Settings

### Connection
- **Protocol**: UDP (faster) or TCP (more stable)
- **Kill Switch**: Blocks internet if VPN drops
- **Split Tunneling**: Choose which apps use VPN

### Privacy
- **DNS Protection**: Prevents DNS leaks
- **IPv6 Leak Protection**: Blocks IPv6 leaks
- **WebRTC Leaks**: Prevents WebRTC IP exposure

### General
- **Auto-Connect**: Connect on startup
- **Notifications**: Alert on connection changes
- **Theme**: Light or dark mode

## Troubleshooting

### Connection Issues

**Can't connect?**
1. Check internet connection
2. Try different server
3. Restart app
4. Check firewall settings

**Slow speeds?**
1. Switch to UDP protocol
2. Select closer server
3. Check WiFi signal
4. Restart router

### Account Issues

**Forgot password?**
1. Click "Forgot Password" on login
2. Check email for reset link
3. Create new password

**Multiple devices?**
1. Log in on each device
2. Use same account credentials
3. Devices share subscription allocation

## Security Tips

1. **Use 2FA**: Enable 2-factor authentication in settings
2. **Strong Password**: Use 16+ character password
3. **Backup Codes**: Save your 2FA backup codes
4. **Update Regularly**: Keep app updated for security patches

## Performance

### Speed Test
1. Connect to VPN
2. Go to Settings → Speed Test
3. Wait for results
4. Check latency, download, upload speeds

### What's Normal?
- Latency: 10-100ms
- Download: 50-500 Mbps (depends on plan)
- Upload: 10-100 Mbps

## FAQ

**Is my data really private?**
Yes. We use military-grade AES-256 encryption. We don't log connection data.

**What's the difference between plans?**
Mainly device count and support level. All plans use same servers and encryption.

**Can I use VPN on routers?**
Yes, download router-specific guides from our support site.

**Does VPN slow internet?**
Slightly, but we optimize for speed. Most users don't notice significant slowdown.

**Is VPN legal?**
Yes, VPNs are legal in most countries. Check local regulations.
